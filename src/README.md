# Паралельне програмування з використанням MPI

## Опис проекту
Цей проект демонструє реалізацію паралельних обчислень з використанням бібліотеки MPI (Message Passing Interface). 
Проект реалізує обчислення формули: MA = MX * MR - min(C) * (MZ * MD)

## Основні компоненти
- `Data` - клас для роботи з даними (вектори та матриці)
- `ThreadBase` - базовий клас для потоків обчислення
- `T1`, `T2`, `T3`, `T4` - класи потоків, що виконують паралельні обчислення

## Особливості реалізації
- Використання MPI для комунікації між процесами
- Розподіл обчислень між 4 процесами
- Синхронізація процесів через бар'єри та обмін повідомленнями
- Вимірювання часу виконання

## Вимоги
- MPI бібліотека
- C++17 або новіше
- Компілятор з підтримкою MPI
